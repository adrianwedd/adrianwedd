# Adrian Wedd — systems architect (Tasmania)

I build evaluation infrastructure for agentic AI and ship the surrounding tooling — MCP servers, voice and embodied interfaces, ops plumbing. Seven years at Tasmania's Department of Communities; wrote Homes Tasmania's first generative-AI policy. No formal degree.

## Now

- **[failure-first-embodied-ai](https://github.com/adrianwedd/failure-first-embodied-ai)** — adversarial evaluation framework for agentic AI. 13,988 scenarios across 414 attack classes; multi-model vulnerability runs.
- **[spark](https://github.com/adrianwedd/spark)** — Claude-powered companion robot on PiCar-X + Raspberry Pi. Built for a neurodivergent kid; declarative-language interaction, deterministic crisis fallbacks.
- **[afterwords](https://github.com/adrianwedd/afterwords)** — zero-shot voice cloning on Apple Silicon. Gives Claude Code a voice. ~15 voices included.
- **[rlm-mcp](https://github.com/adrianwedd/rlm-mcp)** — MCP server implementing the Recursive Language Model pattern. Lets Claude Code work over 1M+ character documents via session-based chunking and BM25.
